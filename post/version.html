규격 정의의 목적

규격을 정의하여 여러 곳에서 쉽게 가져다가 사용할 수 있게 합니다.


규격 규칙

기존항목을 삭제하지 않습니다. (이미 사용하던 곳에서 오류방지)

기존항목을 변경을 하지 않고 유동적으로 사용가능하도록 설계합니다.


V 0.0.2

    {
  • "num":1, => 첫번째 감염자
  • "sex":"w", => 성별 : 여성, 남성은 m {"m","w"}
  • "age":35, => 나이 : 35세
  • "parent":0, => 모 감염자 번호 (기본값은 0)
  • "country":"cn", => cn: 중국, kr: 한국
  • "contact":45, => 접촉자 수
  • "hospital":"인천의료원", => 격리 병원
  • "from":"우한 방문", => 방문 지역 {"국내 감염","?? 방문"}
  • "wuhan":true, => 우한 방문여부
  • "status":1, => 상태 {0:사망, 1:격리 중, 2:완치}
  • "confirmDate":"2020-01-20" => 확진 날짜 {yyyy-mm-dd}
    • }

우한 방문여부를 True,False로 나타내는 항목인 "wuhan" 추가

현재 상태를 표시하는 항목인 status 추가
