소개

신종 코로나 바이러스 감염증의 확진자 현황 관계도를 그린 페이지입니다.

데이터 파일을 이용하여 쉽게 관계도가 그려지도록 만들어졌으며,

누구나 쉽게 데이터 파일에 접근 가능하도록 페이지 하단에 형식을 소개해드립니다.


방문 국가별 데이터를 숨김(▲) / 보이기(▼) 할 수 있습니다.

확진자 클릭시 공식적으로 공개된 정보들이 표시됩니다.

모바일과 PC에서 볼수있으나 모바일은 표시할수있는 양이 적으므로 불편할수있습니다.

데이터 파일

다운로드 : 보기 (.json)

개발자들이 주로 쓰는 json 형식이고 어렵지 않다. 표를 글자로 표현했다고 생각하면 쉽습니다.

형식 (버전 0.0.3) 이전버전 및 규칙 보기

    {
  • "num":1, => 감염자 번호 {1,2,3 ...}
  • "sex":"w", => 성별 : 여성, 남성은 m {"m","w"}
  • "age":35, => 나이 : 35세
  • "parent":0, => 모 감염자 번호 (기본값은 0)
  • "country":"cn", => cn: 중국, kr: 한국
  • "contact":45, => 접촉자 수
  • "hospital":"인천의료원", => 격리 병원
  • "from":"우한 방문", => 모감염자와 관계 또는 방문 지역 {"#0 지인","?? 방문", ...}
  • "wuhan":true, => 우한 방문여부
  • "status":1, => 상태 {0:사망, 1:격리 중, 2:완치}
  • "confirmDate":"2020-01-20" => 확진 날짜 {yyyy-mm-dd}
  • "dischargeDate":"2020-02-06" => 완치 날짜 {yyyy-mm-dd}
    • }

제보

# 제보 / 수정 요청 / 건의 하러가기

만약 반영이 안 된 확진자 정보가 있다면 아래의 형식을 수정하여 수정 요청으로 보내주시면 확인 후 적용해드립니다.

    {
  • "num":1,
  • "sex":"w",
  • "age":35,
  • "parent":0,
  • "country":"cn",
  • "contact":45,
  • "hospital":"인천의료원",
  • "from":"우한 방문",
  • "wuhan":true,
  • "status":1,
  • "confirmDate":"2020-01-20"
  • "dischargeDate":"2020-02-06"
    • }

추가로 표시해줄 내용, 정리해서 보기쉽게 만들 내용이 있다면, 제보바랍니다.
