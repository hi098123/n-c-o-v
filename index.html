데이터 파일

다운로드 : 보기 (.json)

표로 보기 : 표로 보기 (한컴, 워드, 엑셀등에 활용)

개발자들이 주로 쓰는 json 형식이고 어렵지 않다. 표를 글자로 표현했다고 생각하면 쉽습니다.

형식 (버전 0.0.3) 이전버전 및 규칙 보기

    {
  • "num":1,=> 감염자 번호{1,2,3 ...}
  • "sex":"w",=> 성별 : 여성, 남성은 m{"m","w"}
  • "age":35,=> 나이 : 35세
  • "parent":0,=> 모 감염자 번호 (기본값은 0)
  • "country":"cn",=> cn: 중국, kr: 한국
  • "contact":45,=> 접촉자 수
  • "hospital":"인천의료원",=> 격리 병원
  • "from":"우한 방문",=> 모감염자와 관계 또는 방문 지역{"#0 지인","?? 방문", ...}
  • "wuhan":true,=> 우한 방문여부
  • "status":1,=> 상태{0:사망, 1:격리 중, 2:완치}
  • "confirmDate":"2020-01-20",=> 확진 날짜{yyyy-mm-dd}
  • "dischargeDate":"2020-02-06"=> 완치 날짜{yyyy-mm-dd}
    • }
